Introduction to HTML

What is HTML

HTML stands for "HYPER TEXT MARKUP LANGUAGE".

It is the standard language used to create and structure webpages.

Web browser (like Chrome,Firefox,Safari) read HTML code and display it as website

Why learn HTML?

  • It is the foundation of all websites.
  • You need HTML to uild pages before adding CSS (styling) ir Javascript (functionality).
  • Knowing HTML helps you understand how websites work behind the scenes.
